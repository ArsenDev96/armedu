# Translation glossary — ArmEdu

Shared terminology for the three editions. Anything listed here must be used
consistently across every file in that locale. Where a choice was uncertain it is
marked **[review]** and listed in "Items for native review" at the end.

## Orthography rules

**Eastern Armenian (`hy`)** uses **reformed (Abeghian) orthography**, as used in the
Republic of Armenia:

- `-ություն` (պատմություն, գրականություն)
- `և` ligature is used
- `-ված` participles (գրված, հիմնված)
- `ե` / `ո` per reformed spelling

**Western Armenian (`hyw`)** uses **classical (Mesropian) orthography** throughout:

- `-ութիւն` (պատմութիւն, գրականութիւն) — never `-ություն`
- `եւ` written out — never the `և` ligature
- `-ուած` participles (գրուած, հիմնուած) — never `-ված`
- `իւ` where Eastern has `յու`/`ու`; `եա` where Eastern has `յա`
- Classical `է` and `օ` retained (էջ, օր, կ՚ըսէ)
- Present tense with the `կը` particle (կը գրէ, կը կարդայ)
- Infinitives in `-ել` / `-իլ`
- Indefinite `մը` follows the noun (գիրք մը)

Western Armenian is **not** produced by find-and-replacing Eastern Armenian endings.
Sentence structure, verb forms and vocabulary all differ.

## Things that stay in Latin script

- The product name **ArmEdu**
- URL slugs (`tigran-the-great`, `david-of-sassoun`) — shared across all locales
- Filter and category **ids** (`ancient`, `medieval`, `poetry`) — only labels translate
- Email addresses and domains
- Publisher names in `sources` (Cambridge University Press, Matenadaran, …), since
  they identify a specific edition a reader would search for

Source **titles** are translated only when a standard Armenian title exists;
otherwise the original is kept so the work can be found.

## Items for native review

Everything below is used **consistently** across the whole site — the point of this
list is that a native editor can change any entry in one pass. Nothing here is a
known error; these are judgement calls made during translation.

### Eastern Armenian (`hy`)

1. **`Զտիչներ`** for "Filters" — technically correct but reads as software jargon for a
   school audience; `Դասակարգում` may be friendlier.
2. **`Ընթերցվող գրողներ`** for "Popular Writers" — `Հայտնի գրողներ` is more literal.
3. **`Հայոց ամենահայտնի սիրո պատմությունը`** for "Armenia's best-known love story" —
   an editor may prefer `սիրավեպը`.
4. **`տեղապահ`** for "placeholder" in the image caption — reasonable but not settled.
5. **`Ժամանակակից Հայաստան`** for the "Modern Armenia" period — Armenian school
   textbooks more often use `Նորագույն Հայաստան`; `Ժամանակակից` reads as
   "contemporary" and collides with the literary period of the same name.
6. **Century formatting** — Roman numerals (`XIX դար`, `Ք.ա. IX–VI դարեր`) were used per
   Armenian editorial convention. Switch to `19-րդ դար` if house style prefers Arabic.
7. **`ArmEdu-ի խմբագրական խումբ`** for the author byline — `ArmEdu-ի խմբագրություն` is
   equally idiomatic.
8. **`Պարույր Սեւակ`** keeps `եւ` inside the name while reformed `և` is used in running
   prose. Confirm which the project wants for this surname.
9. **Persian and foreign names** — `Հազկերտ Բ`, `Մուշկան Նյուսալավուրտ`,
   `Հռոփանոս Սամոսատցի` have variant spellings in Armenian scholarship.
10. **`ՅՈՒՆԵՍԿՕ`** for UNESCO — some RA style guides prefer `ՅՈՒՆԵՍԿՈ`.
11. **Vardan Mamikonian's line** at Avarayr is given as a modern-Armenian paraphrase to
    match the English register, not as the Grabar original
    («Մահ ոչ իմացեալ՝ մահ է, մահ իմացեալ՝ անմահութիւն է»).
12. **Plural count strings** (`countArticlesOne` / `countArticlesOther`, etc.) are
    intentionally identical: Armenian does not inflect the noun after a numeral. This is
    correct, but looks like a copy-paste bug to a reviewer.

### Western Armenian (`hyw`)

1. **`վայրկեան ընթերցում`** for "min read" — `րոպէ` is also widely used for "minute".
2. **`Փնտռում`** for "Search" (rather than `Որոնում`) — both are attested; `Փնտռում` was
   chosen as the more idiomatic Western form and is used throughout.
3. **`Կայքի քարտէս`** for "Sitemap" — a calque; `Կայքէջի ցանկ` is an alternative.
4. **`Էջին ուղին`** for "Breadcrumb" — no settled Western Armenian term.
5. **`ընտրացանկ`** for "navigation menu" — Eastern-flavoured tech vocabulary; some
   Western sites use `ցանկ`.
6. **`ոչ-ցանկալի նամակ`** for "spam" — many diaspora sites simply keep `սփամ`.
7. **`Թխուկներ`** (cookies) and **`Վերլուծութիւն`** (analytics) are Eastern-origin
   coinages; a diaspora editor may prefer loan forms.
8. **`Երախաներ`** vs `Երեխաներ` for "Children" in the privacy page.
9. **`Բագրատունեաց`** vs `Բագրատունիներու` — the more literary form was chosen.
10. **Era markers** use `Ք.Ա.` / `Ք.Ե.`; some Western publications prefer `Ն.Ք.` / `Յ.Ք.`
    or `Ք.Ծ.Ե.`.
11. **`Սասնայ ծռեր`** for the epic — some Western sources title it `Սասունցի Դաւիթ`.
12. **`Կաթիլ մը մեղր`** for Tumanyan's "A Drop of Honey" uses Western word order; the
    canonical Eastern title is `Մի կաթիլ մեղր`. Confirm whether canonical work titles
    should keep their Eastern form in the Western edition.
13. **`օփերա`** (opera), **`ԵՈՒՆԵՍՔՕ`** (UNESCO) — Western transliterations vary.
14. **`արդիապաշտ`** for "modernist" — `մոդեռնիստ` is also current in diaspora usage.
15. **`հայ-թաթարական բախումներ`** for the 1905–06 clashes — the historical Armenian term;
    an editor may prefer `հայ-ազերիական`.
16. **`Վերադարձ՝ {category}`** — the injected category label cannot be declined, so the
    back-link reads slightly stiff. Grammatically safe.
17. **Charents's line** `Ես իմ անուշ Հայաստանի…` is deliberately left in the poet's
    Eastern original (only `եւ` classicised), since it is a direct quotation.
18. **Register** — the Western edition addresses a diaspora readership with plural-polite
    imperatives (`Ծանօթացէ՛ք`, `Մնացէ՛ք`). Formality level should be checked as a whole.

### Both Armenian editions

- **Section headings in writer biographies** were numbered (`1.`, `2.`, …) to match the
  history articles. The English source leaves the writer headings unnumbered, so the two
  differ in that one respect.
- **`Միքայել Նալբանդյան`** — the English source says "Mkrtich Nalbandian", which appears
  to be an error; the reformer is Mikayel Nalbandian. The Armenian editions use the
  correct name, so they intentionally diverge from the English here.
